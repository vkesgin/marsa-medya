# Frontend - İçerik Planlama

React + Vite + Tailwind ile hızlı bir frontend scaffold'u.

Çalıştırma:

```bash
cd frontend
npm install
npm run dev
```

Önemli çevresel değişkenler (kopyala `.env` olarak):
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- `VITE_API_URL` (ör: `http://localhost:3000/api`)

Not: Login isteği şu anda backend `/api/auth/login` endpoint'ine gönderiliyor. Backend çalışır durumda olmalı.
